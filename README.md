# Audio Steganography in Python

## Description
This project allows you to hide audio files inside images securely using encryption. You can encode an audio file into an image and decode it back later.

## Features
- Encode audio into an image
- Decode audio from an image
- Generate encryption key for security
- GUI built with Tkinter for easy usage

## Technologies Used
- Python 3
- Tkinter (GUI)
- Numpy
- Pillow (Image processing)
- Pygame (Audio playback)
- Cryptography (Encryption)
